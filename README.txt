### **Project Overview**

설화(Seoulhwa)는 서울의 매혹적인 이야기를 하루 동안의 투어와 세심하게 선정된 기념품을 통해 소개하는 플랫폼입니다.

“설화”는 한국의 수도 “서울”과 “이야기”를 의미하는 한국어 “화(話)”를 결합한 이름으로, 현대와 전통이 조화를 이루는 서울의 매력을 전달합니다.

- 설화의 데이 투어는 여행자들이 서울의 문화에 몰입할 수 있도록 설계되었으며, 숨겨진 보석 같은 장소와 상징적인 랜드마크를 포함하여 서울의 다양한 면모를 체험할 기회를 제공합니다.
- 설화의 온라인 쇼핑몰은 전통적 요소와 현대적 감각이 조화롭게 담긴 기념품을 통해 서울과 한국의 이야기를 담아내어 방문객들에게 특별한 경험을 선사합니다.

전통적 요소와 현대적 감각이 조화된 기념품은 방문객들에게 특별한 경험을 선사하며, 각 제품은 한국의 문화와 예술적 유산에 깊이 연결되어 있습니다.

우리의 플랫폼은 외국인 방문객을 주요 대상으로 하여 영어로 제작되었으며, 관리 페이지는 한국어로 제공되어 효율적인 운영을 지원합니다.

### **Technology Stack**

**Frontend**

- **React**: 직관적이고 반응성이 뛰어난 사용자 인터페이스를 위한 프론트엔드 라이브러리.
- **Redux**: 상태 관리를 통해 효율적인 데이터 흐름과 사용자 경험 제공.
- **Tailwind CSS**: 현대적이고 반응형 디자인 구현.

**Backend**

- **Spring Boot**: 확장 가능하고 안정적인 백엔드 애플리케이션 개발.
- **MySQL & JPA**: 데이터 저장과 ORM(Object-Relational Mapping) 관리.

**Deployment & Tools**

- **IIS**: Windows 기반 서버 환경에서 안정적인 웹 애플리케이션 배포를 위한 웹 서버.
- **Tomcat**: Java 기반 애플리케이션을 안정적으로 실행하는 서블릿 컨테이너.
- **React**: 프론트엔드 개발을 위한 주요 도구로 UI/UX 최적화.
- **IntelliJ IDEA**: 개발 효율성과 코드 품질 향상을 위한 고급 IDE.

### **Unique Features**

1. **Immersive Tours**
    
    서울의 매력을 느낄 수 있도록 설계된 투어로, 전통과 현대의 조화를 경험할 수 있는 기회를 제공합니다.
    
2. **Curated Souvenirs**
    
    전통과 현대적 감각이 조화된 기념품을 통해 방문객이 서울과 한국의 감성을 간직할 수 있도록 돕습니다.
    
3. **Interactive Map**
    
    투어 위치를 손쉽게 탐색할 수 있도록 지도 기능을 제공합니다.
    
4. **User-Friendly Design**
    
    외국인 방문객을 위한 직관적이고 접근성 높은 인터페이스 설계.
    
5. **Dynamic Features**
    
    맞춤형 검색 기능으로 개인화된 경험 제공.

### **Our Vision**

설화는 여행자들에게 단순히 장소를 방문하는 경험을 넘어, 서울과 한국의 이야기를 발견하고 기억에 남길 수 있는 플랫폼을 제공합니다.

우리는 기술과 문화적 스토리텔링을 결합하여, 모든 접점에서 품질과 진정성을 전달하는 것을 목표로 합니다.
