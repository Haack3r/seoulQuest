🏙️ Project Overview

- 설화(Seoulhwa)는 서울의 매혹적인 이야기를 하루 동안의 투어와 세심하게 선정된 기념품을 통해 소개하는 플랫폼입니다.

“설화”는 한국의 수도 “서울”과 “이야기”를 의미하는 한국어 “화(話)”를 결합한 이름으로, 현대와 전통이 조화를 이루는 서울의 매력을 전달합니다.

- 데이 투어: 서울의 문화에 몰입할 수 있도록 숨겨진 보석 같은 장소와 상징적인 랜드마크를 포함한 체험 제공.
- 온라인 쇼핑몰: 전통과 현대적 감각이 담긴 기념품을 통해 서울과 한국의 이야기를 담아 특별한 경험을 선사.

우리의 플랫폼은 **외국인 방문객**을 주요 대상으로 하여 영어로 제작되었으며, 관리 페이지는 한국어로 제공되어 효율적인 운영을 지원합니다.


🛠️ Technology Stack

Frontend

- React: 직관적이고 반응성이 뛰어난 UI/UX 개발.
- Redux: 상태 관리와 효율적인 데이터 흐름 제공.
- Tailwind CSS: 현대적이고 반응형 디자인 구현.

Backend

- Spring Boot: 확장 가능하고 안정적인 애플리케이션 개발.
- MySQL & JPA: 데이터 저장 및 ORM(Object-Relational Mapping) 관리.

Deployment & Tools

- IIS: Windows 기반의 안정적인 웹 애플리케이션 배포.
- Tomcat: Java 기반 애플리케이션 실행을 위한 서블릿 컨테이너.
- IntelliJ IDEA: 개발 효율성 향상을 위한 고급 IDE.


🌟 Unique Features

1. Immersive Tours
    
    서울의 매력을 경험할 수 있는 전통과 현대의 조화를 담은 투어 제공.
    
2. Curated Souvenirs
    
    전통과 현대적 감각이 조화된 기념품으로 서울의 감성을 간직.
    
3. Interactive Map
    
    투어 위치와 명소를 손쉽게 탐색할 수 있는 지도 기능.
    
4. User-Friendly Design
    
    직관적이고 접근성 높은 인터페이스 설계.
    
5. Dynamic Features
    
    맞춤형 검색과 다국어 지원을 통해 개인화된 경험 제공.

🌍 Our Vision

설화는 단순히 장소를 방문하는 것을 넘어, 서울과 한국의 이야기를 발견하고 기억에 남는 경험을 선사합니다.

우리는 기술과 문화적 스토리텔링을 결합하여, 모든 접점에서 품질과 진정성을 전달하는 것을 목표로 합니다.
